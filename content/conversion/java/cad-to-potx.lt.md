        Konvertuokite CAD failus į POTX naudodami Java. Tam reikia tik kelių Java kodo eilučių bet kurioje jūsų pasirinktoje platformoje, pvz., „Windows“, „Linux“, „MacOS“.

        Pavyzdžiui, šaltinio CAD failui galite naudoti šias įkėlimo parinktis:

        * automatinis failo formato aptikimas;
        * nurodykite apsaugotų failų slaptažodį (jei failo formatas jį palaiko);
        * pakeiskite trūkstamus šriftus, kad išsaugotumėte dokumento išvaizdą.
        
        Taip pat yra išplėstinių failo POTX konvertavimo parinkčių:

        * konvertuoti konkretų dokumento puslapį arba puslapių diapazoną;
        * pridėti vandenženklį prie konvertuoto POTX.

        Kai konvertavimas bus baigtas, galite išsaugoti POTX failą vietiniame failo kelyje arba bet kurioje trečiosios šalies saugykloje, pvz., FTP, Amazon S3, Google diske, Dropbox ir kt. Atkreipkite dėmesį – norėdami konvertuoti CAD į POTX, jums nereikia diegti jokios papildomos programinės įrangos, tokios kaip MS Office, Open Office, Adobe Acrobat Reader ir kt.

        Paprastas konvertavimas naudojant GroupDocs.Conversion for Java API gali būti atliktas naudojant tik kelias kodo eilutes. Mūsų API palaiko visos pagrindinės platformos ir operacinės sistemos. Prieš vykdydami toliau pateiktą kodą, įsitikinkite, kad jūsų sistemoje yra įdiegtos šios būtinosios sąlygos.

        * Operacinės sistemos: Microsoft Windows, Linux, MacOS
        * Kūrimo aplinkos: NetBeans, Intellij IDEA, Eclipse, etc.
        * Java runtime: J2SE 6.0 and above
        * Gaukite naujausią GroupDocs.Conversion for Java iš [Maven](https://repository.groupdocs.com/webapp/#/artifacts/browse/tree/General/repo/com/groupdocs/groupdocs-conversion)
         
    code: |
        ```java    
        // Įkelti šaltinio failą CAD konvertavimui
          Converter converter = new Converter("input.cad");
          // Paruoškite tikslinio formato POTX konversijos parinktis
          ConvertOptions convertOptions = new FileType().fromExtension("potx").getConvertOptions();
          // Konvertuoti į POTX formatą
          converter.convert("output.potx", convertOptions);
        ```
