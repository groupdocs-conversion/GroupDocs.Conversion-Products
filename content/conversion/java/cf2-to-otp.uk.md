    title_left: "Кроки для перетворення CF2 на OTP у Java"
    content_left: |
        [GroupDocs.Conversion for Java](https://products.groupdocs.com/conversion/java/) дозволяє розробникам легко конвертувати файл CF2 у OTP за допомогою кількох рядків коду.
        
        * Створіть екземпляр класу Converter і надайте файлу CF2 повний шлях
        * Створіть і встановіть ConvertOptions для типу OTP.
        * Викличте метод Converter.Convert і передайте повний шлях та формат (OTP) як параметр

    title_right: "Системні вимоги"
    content_right: |
        Базове перетворення за допомогою GroupDocs.Conversion for Java можна виконати за кілька простих кроків. Наші API підтримуються на всіх основних платформах і операційних системах. Перш ніж виконувати наведений нижче код, переконайтеся, що ваша система відповідає цим передумовам.

        * Операційні системи: Microsoft Windows, Linux, MacOS
        * Середовища розробки: NetBeans, Intellij IDEA, Eclipse, etc.
        * Java runtime: J2SE 6.0 and above
        * Отримайте останню версію GroupDocs.Conversion for Java від [Maven](https://repository.groupdocs.com/webapp/#/artifacts/browse/tree/General/repo/com/groupdocs/groupdocs-conversion)
         
    code: |
        ```java    
        // Завантажте вхідний файл CF2 для конвертації
          Converter converter = new Converter("input.cf2");
          // Задайте параметри перетворення для цільового формату OTP
          ConvertOptions convertOptions = new FileType().fromExtension("otp").getConvertOptions();
          // Перетворіть файл у OTP формат
          converter.convert("output.otp", convertOptions);
        ```
