Pred spustením nižšie uvedeného kódu sa uistite, že máte vo svojom systéme nainštalované nasledujúce predpoklady.

        * Java runtime: J2SE 6.0 and above

    code: |
        ```java    
        // Načítať zdrojový súbor CSV na konverziu
          Converter converter = new Converter("input.csv");
          // Pripravte možnosti konverzie pre cieľový formát HTM
          ConvertOptions convertOptions = new FileType().fromExtension("htm").getConvertOptions();
          // Konvertovať do formátu HTM
          converter.convert("output.htm", convertOptions);
        ```
