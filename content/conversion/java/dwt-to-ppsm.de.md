        Konvertieren Sie Ihre DWT-Dateien in PPSM in Java. Es dauert nur ein paar Zeilen Java-Code auf jeder Plattform Ihrer Wahl, wie Windows, Linux, macOS.

        Beispielsweise können Sie für die Quelldatei DWT die folgenden Ladeoptionen verwenden:

        * automatische Erkennung des Dateiformats;
        * Geben Sie ein Passwort für geschützte Dateien an (sofern das Dateiformat dies unterstützt).;
        * Ersetzen Sie fehlende Schriftarten, um das Erscheinungsbild des Dokuments beizubehalten.
        
        Es gibt auch erweiterte Konvertierungsoptionen für die PPSM-Datei:

        * Konvertieren Sie eine bestimmte Seite eines Dokuments oder eine Reihe von Seiten;
        * füge dem konvertierten PPSM ein Wasserzeichen hinzu.

        Sobald die Konvertierung abgeschlossen ist, können Sie die PPSM-Datei in Ihrem lokalen Dateipfad oder auf einem beliebigen Speicher eines Drittanbieters wie FTP, Amazon S3, Google Drive, Dropbox usw. speichern. Bitte beachten Sie - um DWT zu konvertieren zu PPSM müssen Sie keine zusätzliche Software wie MS Office, Open Office, Adobe Acrobat Reader etc. installieren.

steps:

    title_left: "Schritte zum Konvertieren von DWT in PPSM in Java"
    content_left: |
        [GroupDocs.Conversion for Java](https://products.groupdocs.com/conversion/java/) ermöglicht Entwicklern die einfache Konvertierung der DWT-Datei in PPSM mit ein paar Codezeilen.
        
        * Erstellen Sie eine neue Instanz der Converter-Klasse und laden Sie die Datei DWT mit dem vollständigen Pfad hoch
        * Legen Sie ConvertOptions für den Dokumenttyp auf PPSM fest
        * Rufen Sie die Methode convert() auf und übergeben Sie den Dokumentnamen (vollständiger Pfad) und das Format (PPSM) als Parameter

    title_right: "System Anforderungen"
    content_right: |
        Die grundlegende Konvertierung mit der GroupDocs.Conversion for Java-API kann mit nur wenigen Codezeilen durchgeführt werden. Unsere APIs werden auf allen wichtigen Plattformen und Betriebssystemen unterstützt. Stellen Sie vor dem Ausführen des folgenden Codes sicher, dass die folgenden Voraussetzungen auf Ihrem System installiert sind.

        * Betriebssysteme: Microsoft Windows, Linux, MacOS
        * Entwicklungsumgebungen: NetBeans, Intellij IDEA, Eclipse, etc.
        * Java runtime: J2SE 6.0 and above
        * Holen Sie sich das neueste GroupDocs.Conversion for Java von [Maven](https://repository.groupdocs.com/webapp/#/artifacts/browse/tree/General/repo/com/groupdocs/groupdocs-conversion)
         
    code: |
        ```java    
        // Quelldatei DWT zur Konvertierung laden
          Converter converter = new Converter("input.dwt");
          // Konvertierungsoptionen für das Zielformat PPSM vorbereiten
          ConvertOptions convertOptions = new FileType().fromExtension("ppsm").getConvertOptions();
          // In das PPSM-Format konvertieren
          converter.convert("output.ppsm", convertOptions);
        ```
