Pred spustením nižšie uvedeného kódu sa uistite, že máte vo svojom systéme nainštalované nasledujúce predpoklady.

        * Java runtime: J2SE 6.0 and above

    code: |
        ```java    
        // Načítať zdrojový súbor ODP na konverziu
          Converter converter = new Converter("input.odp");
          // Pripravte možnosti konverzie pre cieľový formát GIF
          ConvertOptions convertOptions = new FileType().fromExtension("gif").getConvertOptions();
          // Konvertovať do formátu GIF
          converter.convert("output.gif", convertOptions);
        ```
