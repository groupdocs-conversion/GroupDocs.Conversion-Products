        Po zakończeniu konwersji możesz zapisać plik PDF w lokalnej ścieżce pliku lub w dowolnej pamięci innej firmy, takiej jak FTP, Amazon S3, Dysk Google, Dropbox itp.           Converter converter = new Converter("input.pcl");
        ```java    
          // Przygotuj opcje konwersji dla formatu docelowego PDF
          ConvertOptions convertOptions = new FileType().fromExtension("pdf").getConvertOptions();
          // Konwertuj na format PDF
          converter.convert("output.pdf", convertOptions);
        ```
