        content: |
          Menggunakan GroupDocs.Conversion untuk .NET, mengkonversi format file dokumen sangat mudah. Contoh berikut menunjukkan cara mengonversi file PDF menjadi file DOC menggunakan C#:  
            
          * Dapatkan jalur file sebagai input
          * Dapatkan jalur penyimpanan file
          * Konversikan file dokumen input
          * Kembalikan output, sebagai jalur atau aliran (sesuai kebutuhan)  
            
            
          ```cs
          ConversionConfig config = new ConversionConfig();
          config.StoragePath = @"D:\storage\"; 
          ConversionHandler conversionHandler = new ConversionHandler(config);
          var convertedDocumentPath = conversionHandler.Convert("sample.pdf", new WordsSaveOptions {  });
          convertedDocumentPath.Save("result-" + Path.GetFileNameWithoutExtension("sample.pdf") + ".doc");
          ```
