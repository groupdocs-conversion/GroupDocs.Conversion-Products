        * Gaukite naujausią GroupDocs.Conversion for .NET iš [Nuget](https://www.nuget.org/packages/groupdocs.conversion)
         
    code: |
        ```csharp    
        // Įkelti šaltinio failą DOCM konvertavimui
          var converter = new GroupDocs.Conversion.Converter("input.docm");
          // Paruoškite tikslinio formato PNG konversijos parinktis
          var convertOptions = converter.GetPossibleConversions()["png"].ConvertOptions;
          // Konvertuoti į PNG formatą
          converter.Convert("output.png", convertOptions);
        ```
