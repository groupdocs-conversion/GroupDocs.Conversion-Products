        * Λειτουργικά συστήματα: Microsoft Windows, Linux, MacOS
        * Περιβάλλοντα ανάπτυξης: Microsoft Visual Studio, Xamarin, MonoDevelop

    code: |
        ```csharp    
        // Φορτώστε το αρχείο προέλευσης DOTM για μετατροπή
          var converter = new GroupDocs.Conversion.Converter("input.dotm");
          // Προετοιμασία επιλογών μετατροπής για τη μορφή στόχου DOTX
          var convertOptions = converter.GetPossibleConversions()["dotx"].ConvertOptions;
          // Μετατροπή σε μορφή DOTX
          converter.Convert("output.dotx", convertOptions);
        ```
