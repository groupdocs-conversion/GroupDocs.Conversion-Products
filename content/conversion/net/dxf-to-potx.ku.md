        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono
        * Nûtirîn GroupDocs.Conversion for .NET ji [Nuget](https://www.nuget.org/packages/groupdocs.conversion) bistînin
         
    code: |
        ```csharp    
        // Ji bo veguhertinê pelê çavkaniyê DXF bar bike
          var converter = new GroupDocs.Conversion.Converter("input.dxf");
          // Vebijarkên veguhertinê ji bo forma armancê amade bikin POTX

          converter.Convert("output.potx", convertOptions);
        ```
