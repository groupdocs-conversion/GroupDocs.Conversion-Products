        แปลงไฟล์ JPF ของคุณเป็น SVG ใน .NET อย่างง่ายดาย คุณสามารถใช้โค้ดได้สองสามบรรทัด C# ในแพลตฟอร์มใดก็ได้ที่คุณเลือก เช่น Windows, Linux, macOS
        คุณสามารถลองแปลงจาก JPF ถึง SVG ได้ฟรี และประเมินคุณภาพผลลัพธ์ของการแปลง นอกจากสถานการณ์การแปลงไฟล์อย่างง่ายแล้ว คุณสามารถลองใช้ตัวเลือกขั้นสูงเพิ่มเติมสำหรับการโหลดไฟล์ซอร์ส JPF และสำหรับการบันทึกผลลัพธ์ SVG 
        
        ตัวอย่างเช่น สำหรับไฟล์ต้นทาง JPF คุณอาจใช้ตัวเลือกการโหลดต่อไปนี้:

        * รูปแบบไฟล์ตรวจจับอัตโนมัติ;
        * ระบุรหัสผ่านสำหรับไฟล์ที่มีการป้องกัน (หากรูปแบบไฟล์รองรับ);
        * แทนที่แบบอักษรที่ขาดหายไปเพื่อรักษาลักษณะที่ปรากฏของเอกสาร.
        
        นอกจากนี้ยังมีตัวเลือกการแปลงขั้นสูงสำหรับไฟล์ SVG:

        * แปลงหน้าเอกสารเฉพาะหรือช่วงหน้า;
        * เพิ่มลายน้ำให้กับไฟล์ SVG ที่แปลงแล้วและอีกมากมาย.

        เมื่อการแปลงเสร็จสิ้น คุณสามารถบันทึกไฟล์ SVG ของคุณไปยังเส้นทางไฟล์ในเครื่องหรือที่เก็บข้อมูลของบุคคลที่สาม เช่น FTP, Amazon S3, Google Drive, Dropbox เป็นต้น โปรดทราบ - หากต้องการแปลง JPF เป็น {{ TO}} ไม่จำเป็นต้องติดตั้งซอฟต์แวร์เพิ่มเติมใดๆ เช่น MS Office, Open Office, Adobe Acrobat Reader เป็นต้น

        * สร้างอินสแตนซ์ของคลาส Converter และจัดเตรียมไฟล์ JPF ด้วยพาธแบบเต็ม
        * สร้างและตั้งค่า ConvertOptions สำหรับประเภท SVG
        * เรียกใช้เมธอด Converter.Convert และส่งเส้นทางและรูปแบบแบบเต็ม (SVG) เป็นพารามิเตอร์

    title_right: "ความต้องการของระบบ"
    content_right: |
        การแปลงพื้นฐานด้วย GroupDocs.Conversion for .NET สามารถทำได้ในไม่กี่ขั้นตอนง่ายๆ API ของเราได้รับการสนับสนุนบนแพลตฟอร์มและระบบปฏิบัติการหลักทั้งหมด ก่อนดำเนินการโค้ดด้านล่าง ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้งข้อกำหนดเบื้องต้นต่อไปนี้ไว้ในระบบของคุณแล้ว

        * ระบบปฏิบัติการ: Microsoft Windows, Linux, MacOS
        * สภาพแวดล้อมการพัฒนา: Microsoft Visual Studio, Xamarin, MonoDevelop
        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono
        * รับ GroupDocs.Conversion for .NET ล่าสุดจาก [Nuget](https://www.nuget.org/packages/groupdocs.conversion)
         
    code: |
        ```csharp    
        // โหลดไฟล์ต้นฉบับ JPF สำหรับการแปลง
          var converter = new GroupDocs.Conversion.Converter("input.jpf");
          // เตรียมตัวเลือกการแปลงสำหรับรูปแบบเป้าหมาย SVG
          var convertOptions = converter.GetPossibleConversions()["svg"].ConvertOptions;
          // แปลงเป็นรูปแบบ SVG
          converter.Convert("output.svg", convertOptions);
        ```
