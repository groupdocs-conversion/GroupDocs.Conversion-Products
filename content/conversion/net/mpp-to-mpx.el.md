        Το [GroupDocs.Conversion for .NET](https://products.groupdocs.com/conversion/net/) διευκολύνει τους προγραμματιστές να μετατρέψουν ένα αρχείο MPP σε MPX με λίγες γραμμές κώδικα.

    title_right: "Απαιτήσεις συστήματος"
    content_right: |
        Η βασική μετατροπή με το GroupDocs.Conversion for .NET μπορεί να γίνει με μερικά απλά βήματα. Τα API μας υποστηρίζονται σε όλες τις μεγάλες πλατφόρμες και λειτουργικά συστήματα. Πριν εκτελέσετε τον παρακάτω κώδικα, βεβαιωθείτε ότι έχετε εγκαταστήσει τις ακόλουθες προϋποθέσεις στο σύστημά σας.

        * Λειτουργικά συστήματα: Microsoft Windows, Linux, MacOS
        * Περιβάλλοντα ανάπτυξης: Microsoft Visual Studio, Xamarin, MonoDevelop
        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono
        * Αποκτήστε το πιο πρόσφατο GroupDocs.Conversion for .NET από το [Nuget](https://www.nuget.org/packages/groupdocs.conversion)
         
    code: |
        ```csharp    
        // Φορτώστε το αρχείο προέλευσης MPP για μετατροπή
          var converter = new GroupDocs.Conversion.Converter("input.mpp");
          // Προετοιμασία επιλογών μετατροπής για τη μορφή στόχου MPX
          var convertOptions = converter.GetPossibleConversions()["mpx"].ConvertOptions;
          // Μετατροπή σε μορφή MPX
          converter.Convert("output.mpx", convertOptions);
        ```
