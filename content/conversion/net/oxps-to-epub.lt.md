        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono
        * Gaukite naujausią GroupDocs.Conversion for .NET iš [Nuget](https://www.nuget.org/packages/groupdocs.conversion)
         
    code: |
        ```csharp    
        // Įkelti šaltinio failą OXPS konvertavimui
          var converter = new GroupDocs.Conversion.Converter("input.oxps");
          // Paruoškite tikslinio formato EPUB konversijos parinktis
          var convertOptions = converter.GetPossibleConversions()["epub"].ConvertOptions;
          // Konvertuoti į EPUB formatą
          converter.Convert("output.epub", convertOptions);
        ```
