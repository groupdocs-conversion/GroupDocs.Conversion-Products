        * Λειτουργικά συστήματα: Microsoft Windows, Linux, MacOS
        * Περιβάλλοντα ανάπτυξης: Microsoft Visual Studio, Xamarin, MonoDevelop
        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono

    code: |
        ```csharp    
        // Φορτώστε το αρχείο προέλευσης PPSX για μετατροπή
          var converter = new GroupDocs.Conversion.Converter("input.ppsx");
          // Προετοιμασία επιλογών μετατροπής για τη μορφή στόχου TXT
          var convertOptions = converter.GetPossibleConversions()["txt"].ConvertOptions;
          // Μετατροπή σε μορφή TXT
          converter.Convert("output.txt", convertOptions);
        ```
