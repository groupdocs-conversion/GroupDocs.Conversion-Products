        به عنوان مثال، برای فایل منبع VSD می‌توانید از گزینه‌های بارگذاری زیر استفاده کنید:

        * رمز عبور فایل های محافظت شده را مشخص کنید (اگر فرمت فایل از آن پشتیبانی می کند);
        * برای حفظ ظاهر سند، فونت های از دست رفته را جایگزین کنید.

        * صفحه سند خاص یا محدوده صفحه را تبدیل کنید;
        * یک واترمارک به فایل تبدیل شده VSTM و بسیاری موارد دیگر اضافه کنید.

        پس از تکمیل تبدیل، می‌توانید فایل VSTM خود را در مسیر فایل محلی یا هر ذخیره‌سازی شخص ثالث مانند FTP، Amazon S3، Google Drive، Dropbox و غیره ذخیره کنید. لطفاً توجه داشته باشید - برای تبدیل VSD به {{ TO}} نیازی به نصب نرم افزار اضافی نیست - مانند MS Office، Open Office، Adobe Acrobat Reader و غیره.

    title_left: "مراحل تبدیل VSD به VSTM در C#"
    content_left: |
        [GroupDocs.Conversion for .NET](https://products.groupdocs.com/conversion/net/) تبدیل فایل VSD به VSTM را با چند خط کد برای توسعه دهندگان آسان می کند.
        
        * یک نمونه از کلاس Converter ایجاد کنید و فایل VSD را با مسیر کامل ارائه دهید
        * ConvertOptions را برای نوع VSTM ایجاد و تنظیم کنید.
        * متد Converter.Convert را فراخوانی کنید و مسیر و قالب کامل (VSTM) را به عنوان پارامتر ارسال کنید.

    title_right: "سیستم مورد نیاز"
    content_right: |
        تبدیل اولیه با GroupDocs.Conversion for .NET را می توان تنها در چند مرحله ساده انجام داد. API های ما در تمام سیستم عامل ها و سیستم عامل های اصلی پشتیبانی می شوند. قبل از اجرای کد زیر، مطمئن شوید که پیش نیازهای زیر را روی سیستم خود نصب کرده اید.

        * سیستم عامل: مایکروسافت ویندوز، لینوکس، MacOS
        * محیط های توسعه: Microsoft Visual Studio, Xamarin, MonoDevelop
        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono
        * آخرین GroupDocs.Conversion for .NET را از [Nuget](https://www.nuget.org/packages/groupdocs.conversion) دریافت کنید
         
    code: |
        ```csharp    
        // فایل منبع VSD را برای تبدیل بارگیری کنید
          var converter = new GroupDocs.Conversion.Converter("input.vsd");
          // آماده سازی گزینه های تبدیل برای قالب هدف VSTM
          var convertOptions = converter.GetPossibleConversions()["vstm"].ConvertOptions;
          // به قالب VSTM تبدیل کنید
          converter.Convert("output.vstm", convertOptions);
        ```
