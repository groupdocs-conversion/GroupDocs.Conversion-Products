        * Λειτουργικά συστήματα: Microsoft Windows, Linux, MacOS
        * Περιβάλλοντα ανάπτυξης: Microsoft Visual Studio, Xamarin, MonoDevelop
        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono

    code: |
        ```csharp    
        // Φορτώστε το αρχείο προέλευσης XLSB για μετατροπή
          var converter = new GroupDocs.Conversion.Converter("input.xlsb");
          // Προετοιμασία επιλογών μετατροπής για τη μορφή στόχου XPS
          var convertOptions = converter.GetPossibleConversions()["xps"].ConvertOptions;
          // Μετατροπή σε μορφή XPS
          converter.Convert("output.xps", convertOptions);
        ```
