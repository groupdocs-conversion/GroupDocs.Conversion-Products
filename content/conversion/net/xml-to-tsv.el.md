        Το [GroupDocs.Conversion for .NET](https://products.groupdocs.com/conversion/net/) διευκολύνει τους προγραμματιστές να μετατρέψουν ένα αρχείο XML σε TSV με λίγες γραμμές κώδικα.
        
        * Δημιουργήστε μια παρουσία της κλάσης Converter και δώστε στο αρχείο XML την πλήρη διαδρομή
        * Δημιουργήστε και ορίστε ConvertOptions για τον τύπο TSV.
        * Καλέστε τη μέθοδο Converter.Convert και περάστε την πλήρη διαδρομή και τη μορφή (TSV) ως παράμετρο

    title_right: "Απαιτήσεις συστήματος"
    content_right: |
        Η βασική μετατροπή με το GroupDocs.Conversion for .NET μπορεί να γίνει με μερικά απλά βήματα. Τα API μας υποστηρίζονται σε όλες τις μεγάλες πλατφόρμες και λειτουργικά συστήματα. Πριν εκτελέσετε τον παρακάτω κώδικα, βεβαιωθείτε ότι έχετε εγκαταστήσει τις ακόλουθες προϋποθέσεις στο σύστημά σας.

        * Λειτουργικά συστήματα: Microsoft Windows, Linux, MacOS
        * Περιβάλλοντα ανάπτυξης: Microsoft Visual Studio, Xamarin, MonoDevelop
        * Frameworks: .NET Framework, .NET Standard, .NET Core, Mono
        * Αποκτήστε το πιο πρόσφατο GroupDocs.Conversion for .NET από το [Nuget](https://www.nuget.org/packages/groupdocs.conversion)
         
    code: |
        ```csharp    
        // Φορτώστε το αρχείο προέλευσης XML για μετατροπή
          var converter = new GroupDocs.Conversion.Converter("input.xml");
          // Προετοιμασία επιλογών μετατροπής για τη μορφή στόχου TSV
          var convertOptions = converter.GetPossibleConversions()["tsv"].ConvertOptions;
          // Μετατροπή σε μορφή TSV
          converter.Convert("output.tsv", convertOptions);
        ```
